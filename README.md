
## 🚀 Deploy on Vercel

### 1 — Push to GitHub
```bash
git init
git add .
git commit -m "Mudra wedding website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/mudra-wedding.git
git push -u origin main
```

### 2 — Deploy on Vercel
1. Go to vercel.com → New Project → Import repo
2. Framework: **Other**
3. Root Directory: leave blank
4. Click **Deploy** — NO env vars needed!

✅ Live at `https://mudra-wedding.vercel.app`
